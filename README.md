## How to publish on GitHub Pages
1. Create a new GitHub repository (public recommended for Pages).
2. Upload all files/folders from this package into the repository root.
3. In GitHub: Settings → Pages → Build and deployment → Source: "Deploy from a branch"
4. Select Branch: `main` and Folder: `/ (root)` → Save.
5. Your site will be available at: `https://<username>.github.io/<repo>/`

## Updating placeholder links
- Replace placeholder `href="#"` items in `modules.html`, `news.html`, and `events.html` with real URLs when ready.
- Add social URLs on `contact.html` once you have them.
